Airway Segmentation & Quantitative CT

Overview

Development of a robust imaging analytics pipeline to characterize respiratory disease phenotypes using quantitative CT. The system performs:

  • High-fidelity airway tree extraction and graph reconstruction
  • Bronchial generation labeling and topology integrity checks
  • Mucus plug detection with volumetric quantification
  • Airway lumen and wall diameter measurements across branches

Motivation

Accurate and automated airway analysis enables stratification of patient subgroups, supports early translational studies, and guides precision medicine decisions.

Technical Highlights

  • Hybrid 3D U-Net + rule-based post-processing for airway continuity
  • Branch graph construction via centerline tracing and radius estimation
  • Mucus candidate classification using intensity + shape + adjacency features
  • Diameter quantification with sub-voxel interpolation (Hermite splines)
  • Modular architecture enabling substitution of segmentation backbone

Impact

  • Accelerated phenotyping workflows for translational bronchiectasis and asthma studies
  • Reduced manual annotation burden for imaging scientists
  • Provided a foundation for a pending patent and manuscript in preparation

Stack / Tools

  • Python (PyTorch, SimpleITK, SciPy, NumPy)
  • Containerized execution (Docker) for reproducibility
  • GPU acceleration for segmentation stage
  • Jupyter notebooks for exploratory QC and visualization

Future Extensions

  • Integration of airway deformable modeling for longitudinal change detection
  • Incorporation of radiomics feature panels for multi-modal correlation
  • Web-based visualization portal (possible Dash/React front-end)
